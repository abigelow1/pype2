Parameters

inputModuleNames
Specify the file names to be included in the docset.

Subclasses should override this method, but should also call the parent method using an algorithm such as: 

          subclass_args = {}
          subclass_args.update( ParentClass.getCloneArguments(
                                    self,
                                    packageName,
                                    baseDirectory,
                                    inputModuleNames) )
          subclass_args.update( {
              "subClassArgument":self._sub_class_argument,
              })
          return subclass_args

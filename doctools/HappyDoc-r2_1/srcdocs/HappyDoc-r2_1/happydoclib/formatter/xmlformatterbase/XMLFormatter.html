  getAttrs  
getAttrs ( self,  attrs )

Converts Python dictionary into XML attributes string. For example

{ 'attr1': val1, 'attr2': val2, 'attr3': val3, }

is converted to the following string

attr1="val1" attr2="val2" attr3="val3"

Arguments

attrs
dictionary with XML attributes

Returns attributes string.

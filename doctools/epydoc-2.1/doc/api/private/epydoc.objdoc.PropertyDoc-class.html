Class PropertyDoc

ObjDoc --+
         |
        PropertyDoc

The documentation for a property. This documentation consists of standard pieces of documentation (as defined in ObjDoc), and the following property-specific pieces of documentation:
  • fget: The property's get function
  • fset: The property's set function
  • fdel: The property's delete function
  • type: The property's type

_make_uid(self, func, cls, functype)

Return a UID for the given fget/fset/fdel function of the property. This is a heuristic function, which first looks for the function within the current class (and its ancestors); and then looks for it as a module-level function. If the function is actually defined somewhere else (e.g., in an unrelated class), then it will be mistakenly interpreted as a module- level function.
Parameters:
func - The function to get a uid for.
cls - The UID for the class containing the property.

Class Param

The documentation for a function parameter. This documentation consists of the following fields:
  • name: The name of the parameter
  • descr: A description of the parameter
  • type: A description of the parameter's type
  • default: The parameter's default value
Params are used by FuncDoc to document parameters.

Method Details

shared_descr_params(self)

Returns:
A list of Params that share the description given by this parameter. This is used in cases where multiple parameters are descrubed in one @param field, such as: 
   @param *varargs, **kwargs: extra args for f.

In this case, the description will be added to varargs's Param; and vararg's Param will list kwargs in its shared_descr_params() list.

For parameters described by normal (single-parameter) @param fields, shared_descr_params will return None.
           (type=None or list of Param)

listed_under(self)

Returns:
The parameter that this Param is listed under, if any. This parameter is listed under parameter p if it contained in p.shared_descr_params().
           (type=None or Param)

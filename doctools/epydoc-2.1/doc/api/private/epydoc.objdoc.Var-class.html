Class Var

The documentation for a variable. This documentation consists of the following fields:
  • uid: The variable's UID
  • descr: A description of the variable
  • type: A description of the variable's type
  • has_value: A flag indicating whether a variable has a value. If this flag is true, then the value can be accessed via the Var's UID.
Vars are used by ModuleDoc to document variables; and by ClassDoc to document instance and class variables.

Class ModuleDoc

ObjDoc --+
         |
        ModuleDoc

The documentation for a module or package. This documentation consists of standard pieces of documentation (as defined in ObjDoc), and the following module-specific pieces of documentation:
  • classes: A list of all classes contained in the module/package.
  • functions: A list of all functions contained in the module/package.
  • variables: A list of all variables contained in the module/package.
  • modules: A list of all modules contained in the package (packages only).
  • imported objects: Lists of imported objects, sorted by type.
